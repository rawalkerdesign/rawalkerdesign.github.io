BEECH GROVE CEMETERY

WEB APP

An Immersive Learning class at Ball State University needed a designer and developer to help them create their app to show different grave plots at the Beech Grove Cemetery. I created all of the UX and UI designs for the app. Additionally, I worked closely along side a class of students to help them translate their ideas into a working web app. The idea behind this project was for visitors of the Beech Grove Cemetery to be able to browse through assorted categories to locate different tombstones. This project was done through my job at the Digital Corps to assist an Immersive Learning class in creating the design layout for a mobile web app.
